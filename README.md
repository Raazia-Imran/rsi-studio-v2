## 📦 Installation & Initialization

To initialize the RSI system locally:

1. **Clone the Repository**:
```bash
git clone https://github.com/raazia-imran/rsi-studio-v2.git
cd rsi-studio-v2

```


2. **Install Dependencies**:
```bash
npm install

```


3. **Run Development Environment**:
```bash
npm run dev

```


4. **Production Build**:
```bash
npm run build

```
